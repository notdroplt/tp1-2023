A biblioteca padrão

Nessa página, daremos uma olhada em algumas funções da bilbioteca padrão de C, sem enrolar mais, aqui estão elas

O printf

O printf é uma função que é capaz de imprimir um texto formatado para a saída padrão, por mais que isso pareça simples, existe uma interessante engenharia por trás de como a função consegue detectar padrões

a lógica é olhar por cada caractere e caso ele seja um caractere de escape, no caso %, acionar as funções formatação

supondo que nos temos as variáveis

// valores aqui estão em média
int tempo_cefet = 7;
int tempo_cefast = 3;
int horas_fazendo_tp1 = 10; // css foi responsável por 90% disso tudo
int tempo_de_sono = 24 - tempo_cefet - tempo_cefast - horas_fazendo_tp1;

podemos colocar no printf a seguinte expressão:

printf("sono durante o tp1: %d horas\n", tempo_de_sono)
sono durante o tp1: 4 horas

Alocação de memória

Alocação de memória é basicamente uma redpill, ela te mostra a verdadeira força da linguagem e permite programas muito mais complexos

No nosso singelo exemplo, assumiremos uma região de memória de 64 bytes e um único ponteiro, para fins de visualização

ao chamar a função malloc, uma certa região da memória será mapeada e alocada para uso arbitrário

Sistema de arquivos, no alcance de uma função

Acessar o sistema de arquivos se torna essencial a partir do momento em que o programa necessita de mais entradas do que a própria entrada padrão

O uso de arquivos é mantido por um "protocolo" de propriedade, onde é responsabilidade do programador abrir, utilizar e fechar um arquivo, na forma

// abrir o arquivo
FILE * fp = fopen("nome do aquivo", "modo");
// exemplo de uso do arquivo
char * str = "um texto pequeno";
fwrite(fp, strlen(str), 1, str);

// fechar o arquivo
fclose(fp);
