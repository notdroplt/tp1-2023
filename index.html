O ABdeC

Nada melhor do que o poder de controlar um computador inteiro com palavras, não é mesmo? Imagine poder controlar TUDO que a máquinatem para oferecer, simplesmente utilizando de uma linguagem que foi aperfeiçoada durante os anos

Esse site é focado em mostrar alguns conceitos por trás da enorme cortina que a linguagem C, junto com o seus compiladores + o ambiente padrão, se envolve para formar um código que é conciso, compreensível e extremamente robusto

Alguns conceitos abordados serão:

  • Os ponteiros
  • A real mágica
  • O famoso malloc
